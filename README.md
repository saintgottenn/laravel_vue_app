

Для работы с проектом убедитесь, что у вас установлены:

- Node.js
- Composer
- PHP
- Система управления базами данных (например, MySQL)

### Установка

Шаги для установки и настройки проекта:

```bash
  git clone https://github.com/saintgottenn/laravel_vue_app
  cd laravel_vue_app
  
  npm install
  composer install
```


Подключите бд(MySQL)

```bash
  php arisan migrate
  php artisan db:seed
```
